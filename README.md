## Descripción

Este proyecto corresponde a mi portafolio web personal, diseñado y desarrollado con enfoque responsivo para adaptarse correctamente a computadores, tablets y dispositivos móviles.

El sitio tiene como objetivo centralizar mi información profesional, mostrando mi experiencia en desarrollo web, programación, bases de datos, tecnologías cloud, ciencia de datos y proyectos realizados.

## Características del portafolio

- Diseño responsivo adaptable a computador, tablet y teléfono.
- Menú de navegación dinámico.
- Sección de presentación profesional.
- Información personal y perfil técnico.
- Barras animadas de habilidades.
- Sección de curriculum con experiencia y formación.
- Galería de proyectos con filtros por categoría.
- Tabla de tecnologías organizada por áreas.
- Botón de descarga de CV.
- Formulario de contacto funcional.
- Integración con GitHub Pages para publicación web.
